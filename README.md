# Weather App 🌦️

A simple Weather Application built using **HTML, CSS, and JavaScript** that fetches real-time weather data using **WeatherAPI**.

## Live Demo 🚀

🔗 https://weather-app-lac-three-85.vercel.app/

## Features

* 🌡️ Temperature in Celsius
* 🤒 Feels like temperature
* 💧 Humidity level
* 🌬️ Wind speed
* ☁️ Weather condition with icon
* 🌍 Search weather by city name

## Technologies Used

* HTML
* CSS
* JavaScript
* WeatherAPI
